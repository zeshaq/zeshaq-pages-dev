This checklist records the Cisco images to download for the lab and the official Cisco download landing URLs. These are not private signed file URLs. Cisco may require login, entitlement checks, export/compliance prompts, and EULA acceptance before downloads appear.

## Rules

- Use an entitled Cisco account.
- Prefer `qcow2` for KVM, libvirt, and EVE-NG.
- Prefer `ova` for VMware.
- Prefer full images over `lite` images for advanced feature testing.
- Do not download EPLD images or physical switch `.bin` images for EVE-NG node boot unless the task is explicitly software-upgrade planning.
- Store downloaded files outside Git.
