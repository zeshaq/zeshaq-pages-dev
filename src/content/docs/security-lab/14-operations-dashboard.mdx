## Triage Order

If EVE-NG or Nexus validation fails:

1. Check EVE-NG reachability and node state.
2. Inspect `tmux ls` for live validation sessions.
3. Review the latest `logs/nexus-*` file.
4. Run the services VRF guard to separate route-leak drift from broader fabric failure.

If SOC daily health fails:

1. Read the generated Markdown health report.
2. Check rule sync, staged file drift, Oxidized status, and backup status first.
3. Reconcile from Git-owned repos unless this is approved break-glass work.

If backup status fails:

1. Check local backup directories on `security-lab-runtime-01`.
2. Check the MinIO/Restic environment file exists on the runtime VM.
3. Re-run the backup-restore pipeline only after confirming secrets are present.
